# Modern AI Ecosystem — Sample Knowledge Base

This sandbox demonstrates tools across the **Modern AI Ecosystem**:

## LLM Layer
- **Cloud**: OpenAI GPT, Anthropic Claude, Google Gemini, Groq
- **Local**: Ollama (Llama 3.1, Mistral), vLLM-compatible APIs

## Agentic AI
- LangGraph multi-step workflows
- CrewAI role-based agents
- Microsoft AutoGen / Semantic Kernel patterns

## RAG Stack
- **Frameworks**: LangChain, LlamaIndex, Haystack, DSPy
- **Embeddings**: OpenAI, Cohere, Voyage, Sentence-Transformers, BGE
- **Vector DBs**: Chroma, Qdrant, PostgreSQL+pgvector, Redis cache

## Memory & Observability
- Redis session memory, Zep (optional)
- LangSmith, Langfuse, Phoenix, TruLens tracing

## Security
- Microsoft Presidio PII detection
- Prompt injection heuristics
- Guardrails AI output validation

## Singapore / PDPA note
Do not index real NRIC, medical records, or personal data in shared sandboxes.
Use synthetic or public documentation only.
